Google Sheets 雲端同步設定教學

步驟一：建立 Google 試算表

  1. 前往 Google Sheets 建立新的試算表
  2. 記下試算表網址中的 ID（網址中 /d/ 和 /edit 之間的部分）

步驟二：建立 Apps Script 專案

  1. 在試算表中點選「擴充功能」→「Apps Script」
  2. 將預設的 Code.gs 內容全部刪除
  3. 複製本系統提供的 Code.gs 程式碼並貼上
  4. 修改程式碼第 19 行的 SPREADSHEET_ID 為你的試算表 ID
  5. 點選「儲存」按鈕

步驟三：部署為網頁應用程式

  1. 點選右上角「部署」→「新增部署作業」
  2. 類型選擇「網頁應用程式」
  3. 執行身分選擇「我」
  4. 存取權限選擇「所有人」
  5. 點選「部署」，並授權存取權限
  6. 複製產生的「網頁應用程式網址」

步驟四：設定本系統

  1. 將複製的網址貼到上方「Apps Script Web App URL」欄位
  2. 點選「測試連線」確認連線成功
  3. 連線成功後，所有調代課紀錄將自動同步到 Google 試算表
注意事項：
  • 若修改 Apps Script 程式碼，需重新部署才會生效
  • 每次重新部署會產生新的網址，需更新本系統設定
  • 試算表中會自動建立「調代課紀錄」工作表
